# Data Structures Assignment

This assignment covers Python data structures through practical exercises with product management scenarios.

## File: `data_structures_assignment.ipynb`

A Jupyter Notebook containing 4 comprehensive tasks on Python data structures.

### Task 1: Product Collections (Lists & Tuples)
**What it does:**
- Creates a list of products
- Defines a sample product tuple with (product_name, price, category)
- Accesses specific elements using indexing
- Appends new products to the list
- Converts tuples to lists, modifies values, and converts back to tuples

**Key concepts:** Lists, Tuples, Indexing, append(), type conversion

---

### Task 2: Categories (Sets)
**What it does:**
- Creates a set of unique product categories (automatically removes duplicates)
- Adds new categories to the set
- Demonstrates that duplicate additions don't affect the set
- Checks membership of categories in the set
- Counts the total number of unique categories

**Key concepts:** Sets, Uniqueness, add(), in operator, len()

---

### Task 3: Product Pricing (Dictionaries)
**What it does:**
- Creates a dictionary mapping product names to their prices
- Adds new products and updates existing prices
- Removes products from the dictionary with error handling
- Calculates the average price across all products
- Finds products with the highest and lowest prices

**Key concepts:** Dictionaries, Key-value pairs, del, sum(), max(), min(), .get()

---

### Task 4: Combined Operations
**What it does:**
- Combines lists, tuples, and dictionaries to create a product catalog
- Creates a catalog as a list of (product_name, price, category) tuples
- Maps categories to lists of products in a dictionary
- Identifies the category with the maximum number of products

**Key concepts:** Data structure combination, Nested structures, Looping, lambda functions

---

## Running the Notebook

1. Open `data_structures_assignment.ipynb` in Jupyter or VS Code
2. Run cells sequentially to execute each task
3. Observe the output to understand how each data structure works
